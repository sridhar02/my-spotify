## My Spotify

This project is built using the spotify developer API 

## Tecnologies Used
- **React** - as a Frontend Framework

## Get Started

- Clone the repo using 
  
```bash
git clone repo-url
```

### Install project dependencies

```bash
npm install
```

### Run application

```bash
npm start
```

this starts the application http://localhost:3000
